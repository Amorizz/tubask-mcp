# Security

This repository contains **public documentation and MCP discovery metadata only**. It does not include application source code, credentials, or deployment configuration.

## Reporting vulnerabilities

If you believe you have found a security issue in the **Tubask hosted service** (`https://tubask.app`), please report it responsibly:

- Email: [support@tubask.app](mailto:support@tubask.app)
- Do **not** open public GitHub issues for undisclosed vulnerabilities

We aim to acknowledge reports within 48 hours.

## What is not in this repo

- API keys, `.env` files, or secrets
- Server implementation or infrastructure configs
- User data or logs

## Service security overview

Public documentation: [tubask.app/docs/security](https://tubask.app/docs/security)
